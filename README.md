## Requirements
- Node >= 20.6.0

## How to use the tool

- Create a local .env, and add the variables from the template, put the name that will go in the signature. Adjust x/y coords as needed.
- ```
  npm install
  ```

- Add pdf files to process into the input folder, an output folder will be generated after processing them.
- Run on the root folder
```
  npm start
```
